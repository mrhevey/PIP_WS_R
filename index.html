    Picture Image Print

    This program allows you to arrange pictures and images on a page to be printed.  Each image can be specifically sized  in a view window.  Once in the view window you can zoom and position the image in the view window to fit your needs.  At any time the page can be saved so the current configuration will not be lost.  The page can either be directly printed or exported as a PDF document.

    Currently under development... More to come soon!
